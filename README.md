# Класифікація котів та собак

Проєкт з предмету **Introduction to Machine Learning for Data Science**

## Опис проєкту

Метою проєкту є створення моделі машинного навчання, яка може розпізнавати зображення котів та собак.

У роботі було виконано підготовку датасету, очищення пошкоджених зображень, експлораторний аналіз даних, навчання двох моделей та порівняння їхніх результатів.

## Використані технології

- Python
- TensorFlow / Keras
- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn
- PIL

## Датасет

У проєкті використано датасет **Cats vs Dogs**, який містить зображення котів та собак.

Початково було 25 000 зображень. Після перевірки було знайдено 2 пошкоджені файли, тому для подальшої роботи використано 24 998 коректних зображень.

Дані були розділені на три частини:

- train
- validation
- test

## EDA

У межах експлораторного аналізу було виконано:

- перевірку кількості зображень у кожному класі;
- візуалізацію прикладів фото;
- аналіз розмірів зображень;
- перевірку дисбалансу класів.

Дисбалансу класів не було: кількість зображень котів і собак була однаковою.

## Моделі

У роботі було побудовано дві моделі:

1. **Simple CNN** — базова згорткова нейронна мережа, навчена з нуля.
2. **MobileNetV2 Transfer Learning** — модель на основі попередньо навченої MobileNetV2.

## Результати

Simple CNN показала точність приблизно **83%** на тестових даних.

MobileNetV2 показала значно кращий результат завдяки використанню transfer learning. Саме цю модель було обрано як фінальну.
